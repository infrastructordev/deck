# Deck API
Infrastructr Deck API.

## Docs

API documentation may be found at [docs.deck-api.infrastructr.dev](https://docs.deck-api.infrastructr.dev).

## Prerequisites

- OpenJDK 11
- Docker >=17.x
- Docker Compose >=2.3

## Local

    $ mvn clean package
    $ docker-compose up -d
